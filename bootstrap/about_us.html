About Us

Personnel Portfolio came into existence when six college students came together and decided to design a simple and easy to use intership finding website. We wanted the website to be easy to use, provide a great user experience, and help students apply to interships quickly. Many of the features and functionality of this website exhibit that characteristics. As time goes on, this website will continue to be updated and additional functionality will be added accordingly.
